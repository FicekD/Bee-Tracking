# Bee tracker

To install required packages run
```
pip install -r "requirements.txt"
```

Run the script with
```
python bee_counter.py
```

Expected directory tree
```
Bee-Tracking
├── data
│  └── 210906_Pokus2
│     └── *.jpg
│  └── 210906_Pokus2_sorted
│     └── *.jpg

```
